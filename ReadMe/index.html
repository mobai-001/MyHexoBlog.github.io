ReadMe - 单页

写在前面

这是一个ReadMe文件,用于介绍单页和注意事项
首先,单页,顾名思义,就是一个个单独的页面,那为什么不是文章呢,因为这一个个页面是有功能的,是一个个我扒来的有趣的(可交互)页面
那为什么我要扒这个呢,因为这也是一种乐趣吧,看着自己日益充实的博客和其中的功能,就是当摆设也很有成就感,这已经是我博客的一部分了
为什么我要单独创建一个页面来描述呢,因为各个单页的功能可能需要介绍
下面就是正文啦,我会一一介绍单页

All In All

所有单页,应该都会有一个配置界面,如果你觉得单页的功能不称心如意,可以去配置界面自定义单页(当然关闭之后要再配置一遍),一般来说单页也包含暂停功能,但只是一般,有时候有的单页不需要暂停功能,还有声音,一般的单页都是没有声音的

单页 - 迷宫

没有声音,没有配置项,每次刷新都会改变迷宫,你需要操控一个善变(鬼畜)的emoji经迷宫到达终点,你可以选择按下键盘上的方向键控制,也可以点击屏幕上的四个按钮,迷宫没有时间限制,享受吧

单页 - 烟花

烟花单页可以说是一个懒人单页,你不需要去点击,只要打开单页,然后看就行了,当然了,你可以点击右上角的设置图标自定义你的烟花之夜(按钮可能很隐蔽),可以调整烟花种类,单个烟花火星密度,烟花质量(只要不是老古董电脑或手机都可以很流畅的开高质量),天空也就是背景的光线亮度,单个烟花的规模(大小),还可以配置是否自动开火(关了你就要手动点击放烟花),结局模式呢,就是会一次性自动放很多烟花(需要打开自动开火),不是结束,也不用重复点击,剩下三个功能就交给小伙伴们去探索啦,自己试试吧,我就不说啦,关闭配置界面,可以发现屏幕上方正中间有个按钮,点击可以打开声音,左上角按钮可以暂停,这些按钮都很隐蔽(划掉)
